## Design Context

Full strategic context lives in [PRODUCT.md](PRODUCT.md); read it before design work.

- **Register:** product (app/panel UI) · **Platform:** adaptive (iOS + Android)
- **Product:** Space Server — cloud file storage for companies (a "corporate Google Drive") that unifies files, signatures, and processes in one panel.
- **Users:** companies and administrative teams.
- **Positioning:** everything in one panel.
- **Personality:** professional and accessible. Anti-reference: dated corporate (heavy navy, text-dense, legacy-system look).
- **Accessibility:** WCAG 2.1 AA baseline.
- **Note:** the code currently in this repo is the marketing landing (web); the app UI described above is not built here yet.
